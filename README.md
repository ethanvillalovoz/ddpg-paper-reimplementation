# ddpg-paper-reimplementation

This repo implements the Deep Deterministic Policy Gradient (DDPG) algorithm from the paper:

> **Continuous Control with Deep Reinforcement Learning**  
> Lillicrap et al., ICLR 2016 ([arXiv:1509.02971](https://arxiv.org/abs/1509.02971))

---

## Overview

DDPG is an actor-critic, model-free reinforcement learning algorithm that can solve continuous control tasks. This implementation follows the original paper.
